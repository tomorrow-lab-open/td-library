# Object Tracking

Reusable MediaPipe object tracking workflows.

This function family is organized by **input source**.

## Available Input Sources

- `camera`
- `kinect`

## Overview

This folder contains MediaPipe-based object tracking workflows using different input sources.

## Typical Outputs

- object position data
- object tracking state
- recognition-related outputs

## Notes

- Output structure may differ depending on input source
- Check each input source folder for setup and dependency details
