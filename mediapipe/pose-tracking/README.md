# Pose Tracking

Reusable MediaPipe pose tracking workflows.

This function family is organized by **input source**.

## Available Input Sources

- `camera`
- `kinect`

## Overview

This folder contains MediaPipe-based pose tracking workflows using different input sources.

## Typical Outputs

- pose landmarks
- body position data
- tracking state

## Notes

- Output structure may differ depending on input source
- Check each input source folder for setup and dependency details
