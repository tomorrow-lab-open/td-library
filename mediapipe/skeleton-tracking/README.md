# Skeleton Tracking

Reusable MediaPipe skeleton tracking workflows.

This function family is organized by **input source**.

## Available Input Sources

- `camera`
- `kinect`

## Overview

This folder contains MediaPipe-based skeleton tracking workflows using different input sources.

## Typical Outputs

- skeleton data
- joint positions
- tracking state

## Notes

- Output structure may differ depending on input source
- Check each input source folder for setup and dependency details
