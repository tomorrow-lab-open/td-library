# Face Tracking

Reusable MediaPipe face tracking workflows.

This function family includes general face tracking workflows.  
Face mesh is documented separately in `mesh.md`.

This function family is organized by **input source**.

## Available Input Sources

- `camera`
- `kinect`

## Overview

This folder contains MediaPipe-based face tracking workflows using different input sources.

## Typical Outputs

- face landmarks
- face position data
- tracking state

## Notes

- Face mesh is documented separately in `mesh.md`
- Output structure may differ depending on input source
- Check each input source folder for setup and dependency details
