# Player Recorder

Reusable Kinect player recorder modules.

This function family is organized by **device version**.

## Available Versions

- `v1`
- `v2`
- `azure`

## Overview

This folder contains Kinect-based player recording workflows for different Kinect systems.

## Typical Outputs

- player presence
- player ID or state
- recorded movement data
- session-based performer data

## Notes

- Output structure may differ between versions
- Use the version folder that matches your hardware
- Check each version README for setup and dependency details
