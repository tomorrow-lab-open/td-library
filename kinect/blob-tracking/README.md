# Blob Tracking

Reusable Kinect blob tracking modules.

This function family is organized by **device version**.

## Available Versions

- `v1`
- `v2`
- `azure`

## Overview

This folder contains Kinect-based blob tracking workflows for different Kinect systems.

## Typical Outputs

- blob positions
- blob area or contour data
- tracking state
- performer presence

## Notes

- Output structure may differ between versions
- Use the version folder that matches your hardware
- Check each version README for setup and dependency details
