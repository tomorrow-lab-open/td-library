# Depth-to-Particles

Reusable Kinect depth-to-particles modules.

This function family is organized by **device version**.

## Available Versions

- `v1`
- `v2`
- `azure`

## Overview

This folder contains Kinect depth-based particle workflows for different Kinect systems.

## Typical Workflow

- Kinect depth input
- depth image processing
- TOP-based conversion
- particle generation

## Notes

- Workflow structure may differ between versions
- Use the version folder that matches your hardware
- Check each version README for setup and dependency details
